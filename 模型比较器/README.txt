================================================================================
                           模型比较器
================================================================================

【功能说明】
  加载两个训练好的模型，使用同一数据集进行对比测试，
  比较两个模型的准确率、各类别表现、错误分析等。

【环境要求】
  Python 3.8+ (建议 Python 3.9 - 3.11)

【安装依赖】
  pip install torch torchvision PyQt5 opencv-python numpy Pillow

【使用方法】
  1. 双击 "启动工具.vbs" 或运行 python model_comparator.py
  2. 分别加载两个要对比的模型文件 (.pth)
  3. 选择测试数据集目录
  4. 点击"开始对比测试"
  5. 查看对比结果

【数据集要求】
  数据集目录结构:
    dataset/
    ├── 角色A/
    │   ├── 001.jpg
    │   └── 002.jpg
    ├── 角色B/
    │   └── 001.jpg
    └── ...

【结果说明】
  - 总体对比: 显示两个模型的整体准确率对比
  - 详细结果: 每张图片的预测结果
  - 错误分析: 两个模型各自的错误案例

【注意事项】
  - 两个模型必须支持相同的分类类别
  - 测试数据集需要包含正确的分类子文件夹
  - 建议使用独立于训练集的测试数据

================================================================================
